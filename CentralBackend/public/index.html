Welcome to the ERP Web Server | API Doc

The ERP Web server is a REST API server that provides a robust backend for managing various aspects for Department of CSE, University of Chittagong . It is built on top of a MySQL database; designed and implemented by Database Administration Team | Level 1 . Read the database documentaion here.
